# learning-algorithms-in-C

Repositorio de exercicios e exemplos em C, organizado para estudo rapido, navegacao limpa e materiais de apoio por atividade.

## Visao rapida

- examples/: exemplos curtos e diretos (scripts isolados).
- activities/: listas de atividades por numero, com codigo, fluxogramas e descritivos.
- docs/: guias de configuracao e uso do VS Code.

## Estrutura detalhada

### examples/

Exemplos simples para testar conceitos ou comandos. Arquivos aqui sao independentes entre si.

### activities/

Cada atividade fica em uma pasta numerada, por exemplo:

- activities/atividade-01/

Dentro de cada atividade:

- questao01.c a questao10.c: codigo-fonte das questoes.
- recursos/: materiais de apoio.

#### activities/atividade-01/recursos/

- fluxogramas/: imagens dos fluxos.
- descritivo/: passo a passo textual das questoes.

## Guias

- Linux: docs/como-programar-c-no-vscode-linux.md
- Windows: docs/como-programar-c-no-vscode-windows.md
- Indice: docs/como-programar-c-no-vscode.md

## Padroes de nomes

- Questao com dois digitos: questao01.c, questao02.c, ...
- Atividade com dois digitos: atividade-01, atividade-02, ...
- Materiais extras sempre em recursos/.
